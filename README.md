# Loan Approval Prediction

## Overview
This project predicts whether a loan application will be approved or rejected using Machine Learning. The model is trained on historical loan application data and provides predictions based on applicant details.

## Features
- Loan approval prediction
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
